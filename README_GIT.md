# gas-pro3 との連携

このフォルダは GitHub の **gas-pro3** リポジトリと連携しています。

## 基本的な操作

### GitHub から最新を取得
```bash
git pull origin main
```

### 変更を GitHub に反映
```bash
git add .
git commit -m "変更内容の説明"
git push origin main
```

### 状態確認
```bash
git status
```
